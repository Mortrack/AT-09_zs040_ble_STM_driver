Detailed Description

GPIO Definition parameters structure.

This contains all the fields required to associate a certain GPIO pin to either the STATE pin of the HM-10 Clone BLE Device Hardware or the MCU pin from which it can be requested to reset the settings of the HM-10 Clone BLE Device.

Note
This structure is meant to be used whenever this module is implemented at the ETX OTA Protocol library made by Mortrack. Therefore, if you do not use this module together with that protocol, feel free to remove this struct in case you want to free memory from your MCU/MPU.
